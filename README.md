# Birkenstock-Sales-Stock-Analysis
Data-driven inventory management dashboard using Power BI to monitor Birkenstock sales, stock levels, and performance trends.
## Project Overview 
This Power BI Dashboard analyzes Birkenstock sales performance and Stock on hand (SOH) across multiple stores.

The Dashboard helps management:
- Monitor store-wise sales quantity
- Compare stock vs average monthly sales
- Identify slow and fast moving products
- Improve inventory planning decisions

## Business Problem 
Birkenstock management was facing:
- No clear visibility of store-wise stock levels
- Overstock in some stores and stock-outs in others
- Difficulty identifying slow-moving products
- No comparison between stock available and actual sales trend

This Resulted in:
- Lost sales opportunities
- High inventory holding costs
- Poor repleinishment planning

## Solution Provided 
This dashboard provides:
- Store-wise Stock vs Sales comparison
- Average Monthly Sales trend
- SOH tracking
- Quantity-Based performance analysis
- Data-driven inventory decision support

## Key KPIs 
- Total Sales Quantity 
- Total Revenue
- Stock On Hand (SOH)
- Average Monthly Sales
- Inventory Coverage

## Tools Used 
- Power BI
- DAX Measures
- Excel Dataset
- Data Modeling (sales & stock tables)
- Line & Clustered Column Chart
